# 🎬 Movie Recommender System

A CLI-based movie recommender system built using Python and Pandas.

## Features
- Search by genre
- Search by rating (min, exact, range)
- Search by year
- Search by movie name
- Similar movie recommendations
- Pagination support

## Dataset
IMDb Top 1000 Movies (1920–2020)

## How to Run

```bash
pip install -r requirements.txt
python src/main.py
